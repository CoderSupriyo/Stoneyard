HOW TO INSTALL FABRIC MOD LOADER AND CARPET MODS?

  1. Download the Fabric Installer
  2. Close Minecraft if it is already running and then double-click the fabric-installer.exe
  3. On the Fabric Installer window click Install
  4. Open minecraft launcher
  5. Log in with your minecraft account credentials if not already. Then at the game version drop down menu, select the new fabric version (release fabric loader .....) and load game.
  6. Open Run or open Start menu and type %AppData% and press enter to search for the location directory.
  7. Then find your .minecraft folder and open it.
  8. You should now see a mods folder. This is the folder where you will install all your other mods.
  9. Download rest of the .jar files ( given below ) and just drag and drop those files inside your mods folder.
  10. After dragging and dropping your .jar files, restart the game to load the mods.
